# JavaGraphGui
### Java FX application that shows the available routes between 12 US cities
1.  The program is a study in unweighted graphs with constructors allowing for ArrayLists or int arrays to create the vertices and edges. 
2.  It is also a study in object oriented design principles utilizing abstract classes, interfaces, etc.
3.  Additionally, the 'Nine Tails Coin Problem' solution is in the application package as it utilizes the abstract graph class
